ULAFF Software Downloads

Step 1

Before moving to the download instructions please make sure you download and install
Google Chrome or Firefox
so that the IPython Notebooks software works properly.

Step 2

Choose your operating system below and follow the instructions:

Windows
Mac
Linux
Web

Step 3

Download notebooks.zip, extract, and replace the notebooks empty directory found in the LAFF directory with the extracted contents.


If you do have trouble after following these instructions, check our Known Issues further down the page or post on Piazza.

Beware that when one has tens of thousands of students, there will be someone out there in the world for whom this does not work. While we will do our best to help you if none of these options work for you, please understand that at some point we have to focus our attention on other parts of the course.

You will still be able to successfully complete the course and learn linear algebra even without being able to do the programming related activities.
